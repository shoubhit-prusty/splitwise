# Splitwise Clone

A full-featured expense splitting web application built to help groups manage shared expenses easily.

## Features

- Track shared expenses between friends and groups
- Automatically calculate who owes whom
- Split bills equally or by custom amounts
- View balances and settle up

## Tech Stack

- HTML, CSS, JavaScript

## Getting Started

Clone the repo and open `index.html` in your browser.

```bash
git clone https://github.com/shoubhit-prusty/splitwise.git
cd splitwise
```
